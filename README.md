#Portfolio Website
Overview

This project is a personal portfolio website developed as part of academic coursework to learn and practice HTML and CSS. The website demonstrates the use of basic web development concepts, page structuring, and styling techniques.

#Objectives

To understand the structure of a web page using HTML

To apply CSS for layout, styling, and responsiveness

To design a simple and professional portfolio layout

To gain hands-on experience in front-end web development

#Technologies Used

HTML5 – for structuring the web pages

CSS3 – for styling, layout, and visual design

#Features

Sidebar navigation

Hero section

About section

Skills section with progress bars

Work count section with background image

Contact section with a form

Responsive layout for different screen sizes

#Conclusion

This portfolio website was created purely for academic learning purposes. It helped in understanding the fundamentals of HTML and CSS and provided practical experience in building a complete static website.
